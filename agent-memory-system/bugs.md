# Bug Log

> **TEMPLATE REQUIREMENT:** Each bug entry must include:
> - **Date** (YYYY-MM-DD)
> - **Brief Description** (1-3 lines)
> - **Root Cause** (what caused the bug)
> - **Solution** (what was fixed)
> - **Prevention** (optional: how to prevent in future)
>
> See [bugs_template.md](references/bugs_template.md) for examples.
